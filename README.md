# Sankofa Data Institute Website

A modern, professional website for Sankofa Data Institute, a data science and analytics training institute focused on Africa.

## About

This website showcases the institute's programs, faculty, community impact, and provides information for prospective students, professionals, and partners.

## Features

- **Clean, Modern Design**: Minimalist aesthetic with African-inspired elements
- **Responsive Layout**: Mobile-first design using Tailwind CSS
- **SEO Optimized**: Built with Next.js for optimal performance and search engine visibility
- **Accessible**: Semantic HTML and ARIA compliance
- **Fast Loading**: Optimized with Next.js and Tailwind CSS

## Pages

- **Home**: Hero section, program previews, impact statistics, and call-to-actions
- **About Us**: Mission, vision, leadership team, and institute story
- **Programs**: Detailed information about data science bootcamp, analytics certification, and corporate training
- **Admissions**: Application process, eligibility requirements, and scholarships
- **Faculty & Mentors**: Instructor profiles and mentorship programs
- **Community & Impact**: Community events, social impact initiatives, and networking opportunities
- **Blog / Insights**: Latest articles on data science trends and African innovation
- **Contact Us**: Contact form, location information, and partnership inquiries

## Technology Stack

- **Framework**: Next.js 16 with App Router
- **Styling**: Tailwind CSS
- **Language**: TypeScript
- **Linting**: ESLint
- **Deployment**: Ready for Vercel, Netlify, or any static hosting

## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser

### Build for Production

```bash
npm run build
npm start
```
